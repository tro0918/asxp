# Roadmap

## Draft 0.2 — package core

- Standard Agent Skills-compatible `SKILL.md`
- `asxp.json` sidecar
- `.asxp` pack, verify, install, and diff
- file integrity and revision
- permission and binding escalation detection
- interactive two-agent demo

## Draft 0.3 — interoperability

- MCP Skills Extension compatibility adapter
- installer adapters for two independent Agent Skills clients
- signature profile and publisher key discovery
- conditional Re-sync reference client
- machine-readable TCK result format

## Draft 0.4 — operational profiles

- minimal `asxp-http` delegation binding
- consent and data-class profile
- audit event schema
- registry metadata mapping

## 1.0 readiness criteria

- two independent implementations
- three compatible host runtimes
- twenty or more normative conformance cases
- public security review and threat model
- documented migration from every draft release
- no unresolved critical interoperability issue
